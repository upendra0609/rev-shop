🚀 QUICK START

1. START YOUR APPLICATION
   mvnw.cmd spring-boot:run

2. PLACE AN ORDER
   curl -X POST http://localhost:8080/api/order/place/1

3. ADD ADDRESS
   curl -X POST http://localhost:8080/api/order/1/address \
     -H "Content-Type: application/json" \
     -d '{"line1":"123 Main St","city":"NYC","country":"USA"}'

4. VIEW HISTORY
   curl http://localhost:8080/api/order/history/1

5. ADMIN DASHBOARD
   http://localhost:8080/admin/orders

6. BUYER ORDERS PAGE
   http://localhost:8080/buyer/orders/1/list
